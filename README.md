## Installation
Here's how to install and add the plugin to mov-cli.

1. Install the pip package.
```sh
pip install mov-cli-films 
```
2. Then add the plugin to your mov-cli config.
```sh
mov-cli -e
```
```toml
[mov-cli.plugins]
films = "mov-cli-films"
```

## Usage
```sh
mov-cli -s films the rookie
```
